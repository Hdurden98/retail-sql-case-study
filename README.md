# Retail SQL Case Study

## Project Overview
This project analyzes retail sales data using SQL to identify trends in revenue, product performance, regional sales, and monthly sales patterns.

## Business Questions
- Which products generate the highest revenue?
- Which regions have the strongest sales performance?
- What months show the highest sales?
- Which product categories perform best?
- What is the average order value?

## Dataset
The dataset contains retail sales information including product names, categories, regions, order dates, sales amounts, and quantities sold.

Dataset file included in this repository:
- retail_sales_data.xlsx

## SQL Queries
The SQL file contains queries used to analyze the dataset, including:

- Total sales by region
- Revenue by product category
- Top-selling products
- Monthly sales trends
- Average order value

SQL file included:
- retail_analysis.sql
